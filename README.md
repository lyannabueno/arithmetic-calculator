### Steps to Run

1. **Clone or download the repository:**

```bash
git clone https://github.com/lyannabueno/arithmetic-calculator.git
```

2. **Install dependencies:**

```bash
npm install
```

3. **Run the project for development:**

```bash
npm run dev
```

4. **Build for production:**

```bash
npm run build
```
